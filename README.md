# 🛒 Blinkit Sales Analytics Dashboard

An interactive Power BI dashboard developed to analyze Blinkit's retail sales performance and uncover actionable business insights through dynamic visualizations and KPIs.

## 🚀 Features

- Interactive dashboard with dynamic slicers and filters
- Sales, Average Sales, Item Count, and Rating KPIs
- Outlet performance and location analysis
- Product category and fat content analysis
- Sales trend by outlet establishment year
- Data transformation using Power Query
- Business metrics using DAX

## 🛠️ Tech Stack

- Power BI
- DAX
- Power Query
- Microsoft Excel
